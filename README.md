# Image to image comic translate 

## Quick start

1. **Create a virtual environment**
   ```bash
   python3 -m venv .venv
   source .venv/bin/activate
   python -m pip install --upgrade pip
   ```
2. **Install notebook essentials**
   ```bash
   pip install -r requirements.txt
   ```
   > ⚠️ Install PyTorch with the wheel that matches your platform by following the instructions at [pytorch.org](https://pytorch.org/get-started/locally/). Once you have the correct command, run it inside the activated virtual environment.

for osx now it's
`pip install torch torchvision`

4. **Launch Jupyter**
   ```
   jupyter notebook
   ```
